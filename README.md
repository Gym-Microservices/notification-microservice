# 📧 Notification Microservice

## 📋 Descripción

Microservicio para el envío de notificaciones del gimnasio (Puerto 8085).
Permite enviar notificaciones personalizadas a usuarios específicos del sistema.

## 🔗 Endpoints

### 📨 Gestión de Notificaciones

- `POST /api/notifications` - Enviar una notificación

## 🛠️ Tecnologías

- Spring Boot
- Spring Data JPA
- H2 Database (en memoria)
- Eureka Client
